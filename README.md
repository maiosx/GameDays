Introduction

GameDays is a game progress tracker made by Claude created using the app templates from HabitKit and JustMe. a discord style note taking app.

Claude Chat log

Installation

Simply download the required dependency 
-nodejs

https://nodejs.org/en/download
Then run GameDays.bat

or use the accompanied GameDays.html file and set it as a web app icon on your taskbar or pin it on your web browser.

Usage

To enter a session entry type /game followed by it's name, then your progress note.
